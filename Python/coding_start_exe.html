＞一斉起動(exe化)



コーディング時に使用するツールをまとめて起動する機能を、

Pyinstallerでexe化。






import webbrowser
import subprocess

#ChatGPTを開く
webbrowser.open('https://openai.com/blog/chatgpt')

#サクラエディタを開く
subprocess.Popen(r"C:\Program Files (x86)\sakura\sakura.exe")

#VSCodeを開く
subprocess.Popen(r"C:\Users\YOHEI\AppData\Local\Programs\Microsoft VS Code\Code.exe")

#Appフォルダを開く
subprocess.Popen(['explorer', r"C:\Users\YOHEI\App"], shell=True)




#exe化
pyinstaller coding_start.py --onefile --noconsole --icon=favicon.ico
